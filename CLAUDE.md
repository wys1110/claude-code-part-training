# Claude Code Training Site

## Purpose
- Korean training material for learning Claude Code concepts, useful features, and hands-on workflows.
- Keep examples generic and safe for a public repository.

## Commands
- Install dependencies: `npm ci`
- Run docs locally: `npm run docs:dev`
- Build docs: `npm run docs:build`
- Run lab tests: `cd labs/task-board && npm test`

## Rules
- Do not add company-confidential names, code, URLs, credentials, or internal processes.
- Prefer official Claude Code documentation as the source of product behavior.
- Keep beginner-facing explanations concise and action-oriented.
- Before completing a change, run the docs build and relevant lab tests.
- The lab intentionally begins with failing tests; do not silently fix it unless the task requests it.
